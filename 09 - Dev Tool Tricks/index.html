Dev Tool Tricks

Time for deliberate practice with Dev Tools! Open up your Chrome console to work through these examples.

  1. Break on attribute modification - a technique for finding the JavaScript that is causing a change to an element

    Right-click on this paragraph and inspect it. Right-click again on the highlighted paragraph in the Elements tab. Select `Break on > attribute modifications`. Then click on this paragraph on the page and you should now be running the debugger. Sweet!

  2. Console.log() with string interpolation: (Though you may prefer ES6 template literals)

    console.log('I am a%s string.', 'n interpolated');
  3. Console.log() with styled output: 

    console.log('%cI am some styled text.', 
            'font-size: 50px; background: green;');
  4. Console.warn():

    console.warn('Uh oh, a warning!');
  5. Console.error():

    console.error('Oh shit, an error!');
  6. Console.info():

    console.info('Fun fact: You just printed some info to the console.');
  7. Console.assert() for testing:

    console.assert(1 === 2, 'The two numbers are not equal.');
  8. Console.dir() to see a dropdown list of all the properties available to an element:

    console.dir(para);
  9. Console.count(): 

    console.count('Red team');

    console.time('fetching data');

  11. Console.table() for improved readability:

    console.table(anArray);
